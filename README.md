## 📝 **Project Overview (Legacy Version)**

> **Basic Python-based virtual assistant with fundamental voice command capabilities. This version provides essential desktop automation features for learning and basic productivity tasks.**

**Supriya** is an introductory virtual assistant built with Python that demonstrates core concepts of speech recognition, text-to-speech, and desktop automation. While functional, this version serves as a foundation for understanding AI assistant development.

## 🎯 **Usage Examples (Legacy)**

<div align="center">

### 🗣️ **Voice Commands**

| Category | Command Example | Expected Response |
|:---------|:---------------|:------------------|
| **Application** | *"Open YouTube"* | Launches YouTube in default browser |
| **Search** | *"Search Wikipedia for Python"* | Reads Python article summary |
| **System** | *"Take a screenshot"* | Captures and saves screenshot |
| **Entertainment** | *"Tell me a joke"* | Shares a random joke |
| **Weather** | *"What's the weather"* | Basic weather information |

</div>

### 📝 **Basic Usage Flow:**

1. **🎤 Start Assistant**: Run the Python script
2. **👂 Listen Mode**: Assistant waits for "Hey Supriya" wake word
3. **🗣️ Give Command**: Speak your command clearly
4. **🔄 Processing**: Assistant processes and executes command
5. **📢 Response**: Gets audio/visual feedback

### ⚠️ **Legacy Limitations:**
- **🎯 Limited Accuracy**: Voice recognition may require multiple attempts
- **⏱️ Slow Response**: Processing takes longer than modern assistants
- **🔊 Basic Audio**: Simple text-to-speech without natural intonation
- **📱 Platform Issues**: May not work consistently across all systems
